# haackathon-game-project: QuickPicks
Overview;

QuickPicks is a live football betting/prediction game built for the Superteam x TxLINE hackathon.

Players create their own bets by combining prediction cards, choosing a team, and selecting a time window (5 or 10 minutes). The game then uses live TxLINE match data to determine whether the prediction succeeds.

Unlike traditional betting apps with predefined markets, QuickPicks allows players to build creative predictions from multiple events.

Features;

Live football data powered by TxLINE
Card-based prediction builder
Team selection
Risk/reward time mechanic (5-minute and 10-minute bets)
Dynamic fixture loading
Automatic team name loading from live fixtures

Tech Stack;

GDevelop 5
JavaScript (API integration)
TxLINE Sports Oracle API

TxLINE Endpoints Used;

Guest Session Authentication
Fixture Snapshots
Latest Score Event Snapshots

Repository Structure;

/source files -> Gdevelop project
/docs -> HTML export (for Itch.io)

Built For;
